# Umrah-planner

iOS project created on Hackhathon organized by Hickmet Travel, Sajda App, Nur-Mubarak University and Bahandi. With over 100 participants and 30+ teams from across the region.


## Idea
The GOAL is to make the journey of Umrah accessible to all Muslims.
1. Save time and money
2. Full plan of your journey
3. Support for users with different budgets

## Technologies Used

- Swift
- SwiftUI
- OpenAI LLM
- UI/UX desgin
